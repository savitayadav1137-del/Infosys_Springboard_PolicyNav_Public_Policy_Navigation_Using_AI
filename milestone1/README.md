# Milestone 1 – User Authentication System

## Project Title
PolicyNav – Public Policy Navigation Using AI

## Internship
Infosys Springboard Virtual Internship 6.0 (Batch 13)

## Description
This milestone implements a secure user authentication system using Streamlit, SQLite, and JWT.
The system allows users to sign up, log in, reset passwords using a security question, and access
a protected dashboard.

## Technologies Used
- Python
- Streamlit
- SQLite Database
- JWT (JSON Web Token)
- Ngrok (for public URL demo)

## Features Implemented
- User Signup with validations
- Secure Login using JWT
- Dashboard after login
- Forgot Password using Security Question
- Logout functionality
- Ngrok integration for live demo

## How to Run the Application
1. Install required libraries:
pip install streamlit pyjwt
2. Run the Streamlit application:  
streamlit run app.py
3.  Expose the application using ngrok (for live demo)
ngrok http 8501
